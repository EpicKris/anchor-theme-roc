## Site Variables

### Profile Picture
Name: `profile_picture`

Value: `instagram`, `twitter`, or image URL.

### Instagram
Name: `instagram`

Value: Your Instagram account (for example; `EpicKris`).

###`twitter`
Name: `twitter`

Value: Your Twitter account (for example; `EpicKris`).
